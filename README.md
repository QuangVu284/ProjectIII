# ProjectIII

## Getting Started

### Prerequisites

- [Install NodeJS ](https://nodejs.org/en/download)
- Install NPM

### Installation

1. Clone the repository https://github.com/QuangVu284/ProjectIII.git
2. Install NPM packages

```
$ cd frontend
$ npm i --force
```

```
$ cd backend
$ npm i
```

## Usage

1. Run the React scripts

```
$ cd backend
$ npm start
```

```
$ cd frontend
$ npm start
```
